# 🎓 מרכז תרגול – תנאים ולולאות

משימה כיתתית לשיעור 4: בניית דף אינטרנט אינטראקטיבי שמשתמש בתנאים, לולאות ופונקציות ב-JavaScript.

## 📋 תיאור הפרויקט

פרויקט זה כולל 5 תרגילים מעשיים שמאפשרים לתלמידים לתרגל:
- שימוש בלולאות (`for` ו-`while`)
- שימוש בתנאים (`if` ו-`switch`)
- כתיבת פונקציות שמחזירות ערכים
- עבודה עם מערכים
- אינטראקציה עם DOM

## 🚀 איך להפעיל

1. שכפלו או הורידו את הפרויקט
2. פתחו את הקובץ `index.html` בדפדפן
3. פתחו את ה-Console (F12 → Console) כדי לראות חלק מהתוצאות
4. עקבו אחר ההוראות בקובץ `main.js` כדי להשלים את המשימות

### 2️⃣ מחשבון הודעות ציון
- **מטרה**: הצגת הודעה מתאימה לפי ציון שנבחר (A-F)
- **כלים**: `switch` statement, פונקציות

### 3️⃣ סכום מספרים שמתחלקים ב-3 או 5
- **מטרה**: חישוב סכום כל המספרים שמתחלקים ב-3 או 5 עד מספר מסוים
- **כלים**: לולאת `for`, תנאי `if`, אופרטורים לוגיים

### 4️⃣ חיפוש מספר במערך עם while
- **מטרה**: חיפוש המספר 42 במערך נתון
- **כלים**: לולאת `while`, עבודה עם מערכים

### 5️⃣ סינון מספרים זוגיים ממערך
- **מטרה**: יצירת מערך חדש עם כל המספרים הזוגיים ממערך נתון
